# 🚀 QUICK START GUIDE

## Step 1: Install Dependencies
```bash
pip install flask
```

## Step 2: Start the Bot
```bash
python app.py
```

Or double-click: `start_bot.bat` (Windows)

## Step 3: Open Browser
Go to: **http://localhost:5000**

## Step 4: Start Chatting!

### Try These Questions:

**Basic:**
- "What is the status of Team A?"
- "Tell me about Team W"

**Formations:**
- "What formation does Team A use?"
- "Show me all formations"

**Line Positions:**
- "Where was the back line?"
- "Show me defensive line positions"

**Comparisons:**
- "Compare both teams"
- "Who attacked more?"

**Comprehensive:**
- "Tell me everything about Team A in build up"

## 🎯 That's It!

The bot now understands **ANY** question you ask about the data!
